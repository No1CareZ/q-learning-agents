# q-learning-agents

3 impleminattions of q-learning agents

## Запуск

Для запуска требуется Python версии 3.12.5

```bash
pip install -r r.txt
```

После этого можно спокойно запускать один из 2 скриптов:

* main.py - запуск без создания визуализации
* main2.py - запуск с созданием визуализации и tqdm
